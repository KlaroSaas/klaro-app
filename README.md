# Klaro

Plateforme minimaliste pour héberger et vendre une formation en ligne.

## Stack
- Next.js 14 (App Router) + TypeScript + Tailwind CSS
- Supabase (base de données + auth)
- Stripe (paiements)
- Hébergement : Vercel

## Démarrage local (optionnel)
```bash
npm install
cp .env.example .env.local   # puis remplir les valeurs
npm run dev
```

## Déploiement
Ce projet est prévu pour être déployé automatiquement via Vercel,
connecté à ce dépôt GitHub. Voir les instructions fournies par Claude
dans la conversation pour la marche à suivre pas à pas.
